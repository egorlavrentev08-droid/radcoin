## 🌟 Основные возможности

| Механика | Описание |
|----------|----------|
| 💰 **Сбор ресурсов** | Добыча RC, RF, кристаллов с учётом локаций, классов и экипировки |
| 🏹 **Охота** | Бои с мутантами и классовая война (военные → бандиты → сталкеры) |
| 🚇 **Метро** | Хардкорное подземелье с пошаговыми боями и аптечками из рюкзака |
| 🔬 **Лаборатория** | Уникальная механика для класса "Учёный" |
| 🎒 **Рюкзаки** | 3 уровня, бонусы к сбору, хранение аптечек для спасения |
| ⚡ **Энергетики** | 4 уровня (Strike/Tornado/Adrenaline/RedBull) с временными бонусами |
| ⏱️ **Редукторы** | 3 уровня (Базовый/Продвинутый/Квантовый) для ускорения сбора |
| 🔍 **Металлоискатель** | Поиск сундуков с кулдауном |
| 🎭 **Классовая система** | Сталкер, Военный, Бандит, Учёный — каждый со своими бонусами |
| 🏰 **Кланы** | Создание, улучшения, казна, клановые кристаллы |
| 🎁 **Сундуки** | 5 типов с уникальными наградами (оружие, броня, питомцы, рюкзаки) |
| 🐾 **Питомцы** | 6 видов с разными бонусами, шанс встретить при сборе |
| 📻 **Радио** | Система оповещений и массовых сообщений |
| 🎰 **Казино** | Настраиваемые шансы и множители |
| 💾 **Автобэкапы** | Каждые 15 минут, автоматическое восстановление при повреждении БД |

## 🆕 Новое в версии 4.0 (ALPHA)

### ✨ Добавлено
- **4 уровня энергетиков** (Strike, Tornado, Adrenaline, RedBull)
- **3 уровня редукторов** (Базовый, Продвинутый, Квантовый)
- **3 уровня рюкзаков** (Маленький, Тактический, Профессиональный)
- **Металлоискатель** — поиск сундуков с кулдауном
- **Лаборатория** (`/lab`) — уникальная механика для учёных
- **Метро** (`/metro`) — хардкорное подземелье
- **Пошаговые бои** с использованием аптечек из рюкзака
- **Классовая война** в охоте
- **Автоматическая миграция БД** при обновлении
- **200+ атмосферных сообщений** для всех действий

### 🔧 Изменено
- Полная переработка охоты (`/hunt`)
- Обновлён магазин — новые товары, убраны лимиты
- Обновлён инвентарь — отображение рюкзака и аптечек
- Расширена система крафта

## 📖 Команды бота

### Основные команды
| Команда | Описание |
|---------|----------|
| `/start` | Начать игру |
| `/help` | Справка |
| `/profile` | Профиль игрока |
| `/stats` | Статистика |
| `/achievements` | Достижения |

### Ресурсы и охота
| Команда | Описание |
|---------|----------|
| `/collect` | Сбор ресурсов |
| `/hunt` | Охота на мутантов |
| `/metro` | Хардкорное подземелье (10+ уровень) |
| `/locate [название]` | Смена локации |
| `/lab` | Лаборатория (для учёных) |

### Экономика
| Команда | Описание |
|---------|----------|
| `/shop` | Магазин |
| `/buy [товар] [кол-во]` | Купить предмет |
| `/sell [предмет] [кол-во]` | Продать предмет |
| `/equip [тип] [название]` | Экипировать предмет |
| `/inv` | Инвентарь |
| `/use [предмет] [кол-во]` | Использовать предмет |
| `/casino [сумма]` | Казино |
| `/exchange [количество]` | RF → RC |
| `/craft [предмет]` | Крафт |

### Классы и эффекты
| Команда | Описание |
|---------|----------|
| `/class [название]` | Сменить класс |
| `/class_info` | Информация о классе |
| `/effect` | Активные эффекты |
| `/effect_clear` | Очистить эффекты |

### Кланы и радио
| Команда | Описание |
|---------|----------|
| `/clan [команда]` | Управление кланом |
| `/radio [текст]` | Отправить в радиоэфир |
| `/radion [код]` | Активировать радио |

### Сундуки и фабрики
| Команда | Описание |
|---------|----------|
| `/chest list/chance/open` | Управление сундуками |
| `/factory [команда]` | Ресурсные точки |

### Прочее
| Команда | Описание |
|---------|----------|
| `/top [категория]` | Таблица лидеров |
| `/advice [раздел]` | Советы |
| `/announce on/off` | Уведомления |
| `/phase_info` | Фаза Пустоши |
